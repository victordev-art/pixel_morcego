# 🦇 Morcego Pixel Art

Animação pixel art de um morcego voando feita com HTML, CSS e JavaScript puro.

## O que aprendi

- Como usar o `canvas` do HTML para desenhar
- Como o JavaScript se conecta ao HTML
- Loops `for` para repetir código
- Loop aninhado para percorrer linhas e colunas
- Como criar frames e animar com `setInterval`
- Como centralizar elementos na tela com CSS

## Como rodar

1. Clone o repositório
2. Abra o `morcego.html` no navegador

## Tecnologias

- HTML
- CSS
- JavaScript
